# lab3_java
Реализовать сериализацию данных в файл на диске для Задания №1 в виде отдельного класса с методами Save и Load. Использовать 2 способа: внешнюю библиотеку и нативную сериализацию. Предусмотреть автоматическое создание резервной копии файла данных при каждом запуске программы и восстановления данных из созданной копии при запуске.
